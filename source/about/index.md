# About This website

## TAROS
本站的名字，意思源于全金属狂潮的全意识控制的人机交互系统。巧的是也有野芋的意思，是我生长的地方最常见的植物之一。被誉为地里的荷叶，茎干部分还可以做菜吃。

## 为何做博客？
分享技术的博客，目前主要方向是前端。为何要做自己的站点？刚开始觉得新鲜好玩，一方面也是想提高写作能力和持续学习输出，另一方面是觉得在分享很重要。茫茫中找到一篇能够解决问题的文章，就像在沙漠中找到水井一般。
